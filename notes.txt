/drive
  /{folder}
    - GET          : Get all the image
    /{file}
      - POST      : Upload the file
      - DELETE    : Delete the file
      - PUT       : Update the image data

/folders
  - POST
  /{id}:
    - DELETE
    - PUT
  /{parentId}:
    -GET

/notes
  - GET           : Get all notes
  /{id}:
    - GET         : Get note for an id
    - POST        : Add note for an id
    - PUT         : Update note for an id
    - Delete      : Delete note for an id

Response code
code 0 : No message
code 1 : show Success
code 2 : show Error
code 3 : show both
